## Setting Up Your Environment

### **1. Install Essential Tools**

#### **Code Editor**

- **Cursor** (Recommended for AI assistance)
- **VS Code** (Alternative)
- **Sublime Text** (Lightweight option)

#### **Development Tools**

- **Node.js** - JavaScript runtime
- **Git** - Version control
- **Terminal** - Command line interface

### **2. Create Your Workspace**

```bash
# Create a folder for your projects
mkdir cardano-projects
cd cardano-projects

# Create your first project
mkdir my-first-dapp
cd my-first-dapp
```

### **3. Set Up Version Control**

```bash
# Initialize Git repository
git init

# Create your first commit
git add .
git commit -m "Initial commit: Starting my Cardano journey"
```

## Understanding the Cardano Ecosystem

### **Core Components**

- **Cardano Node** - The blockchain itself
- **Wallets** - Store and manage ADA
- **Smart Contracts** - Programmable logic
- **dApps** - Decentralized applications

### **Development Tools**

- **MeshJS** - JavaScript SDK for Cardano
- **Plutus** - Smart contract language
- **Cardano CLI** - Command line tools
- **Blockfrost** - API for blockchain data

### **Networks**

- **Mainnet** - Real Cardano network (real money)
- **Testnet** - Testing environment (fake money)
- **Devnet** - Development environment

## Your First Project: Hello Cardano

### **Project Overview**

We'll build a simple web page that:

- Displays Cardano information
- Shows current ADA price
- Explains blockchain concepts
- Demonstrates basic web development

### **Step 3: Add Interactivity**

```javascript
// script.js
document.addEventListener("DOMContentLoaded", function () {
  // Display current date
  const dateElement = document.createElement("p");
  dateElement.textContent = `Today is: ${new Date().toLocaleDateString()}`;
  document.querySelector("header").appendChild(dateElement);

  // Simulate ADA price (in real app, you'd fetch from API)
  const adaPrice = document.getElementById("ada-price");
  adaPrice.innerHTML = `
        <div style="font-size: 2rem; font-weight: bold;">
            $0.45 USD
        </div>
        <small>Updated: ${new Date().toLocaleTimeString()}</small>
    `;

  // Add interactive elements
  const sections = document.querySelectorAll("section");
  sections.forEach(section => {
    section.addEventListener("click", function () {
      this.style.transform = "scale(1.02)";
      setTimeout(() => {
        this.style.transform = "scale(1)";
      }, 200);
    });
  });
});
```

## Common Challenges & Solutions

### **"I don't understand the code"**

- **Solution**: Use ChatGPT to explain each line
- **Practice**: Modify small parts and see what happens
- **Remember**: You don't need to understand everything at once

### **"My project doesn't work"**

- **Solution**: Check browser console for errors
- **Practice**: Start simple and add features gradually
- **Remember**: Debugging is a normal part of development

### **"I'm overwhelmed by all the information"**

- **Solution**: Focus on one concept at a time
- **Practice**: Build small projects to reinforce learning
- **Remember**: Everyone starts as a beginner
