# What is AI?

Welcome to the world of artificial intelligence! If you're completely new to AI, don't worry - we're going to break this down into simple, understandable pieces.

## What is Artificial Intelligence?

Artificial Intelligence (AI) is technology that enables computers to perform tasks that typically require human intelligence. Think of it as giving computers the ability to:

- **Learn** from examples and data
- **Reason** and solve problems
- **Understand** language and images
- **Create** new content like text, images, or code

## Why AI Matters for Cardano Development

Building on Cardano doesn't mean you have to do everything manually. AI can help you:

- **Write code faster** - Generate boilerplate and common patterns
- **Debug issues** - Get instant help with error messages
- **Learn new concepts** - Break down complex topics into simple explanations
- **Build better projects** - Get suggestions for improvements and best practices

## Types of AI You'll Use

### 1. **ChatGPT** - Your Coding Assistant

- Ask questions about Cardano development
- Get help with code problems
- Learn new concepts step by step

### 2. **Cursor** - AI-Powered Code Editor

- Write code with natural language
- Get instant suggestions and completions
- Debug and refactor code automatically

### 3. **Windsurf** - AI Development Environment

- Build entire projects from descriptions
- Get real-time coding assistance
- Collaborate with AI on complex features

## Common AI Terms

- **Prompt** - The instructions you give to AI
- **Model** - The AI system that processes your requests
- **Fine-tuning** - Training AI on specific types of content
- **Hallucination** - When AI makes up information (we'll teach you to avoid this!)
